01. 자식 노드 탐색

노드 탐색

공백 텍스트 노드

        HTML 요소 사이의 스페이스, 탭, 줄바꿈(개행) 등의 공백 문자는 텍스트 노드를 생성한다.

자식 노드 탐색

  1. Node.prototype.childNotes : 자식노드(요소 노드, 텍스트 노드)를 탐색하여 NodeList에 댐아 반환
  2. Node.prototype.firstChild : 첫번째 자식 노드(요소 노드, 텍스트 노드) 반환
  3. Node.prototype.lastChild : 마지막 자식 노드(요소 노드, 텍스트 노드) 반환
  1. Elemnet.prototype.children : 자식 노드 중 요소 노드만 탐색하여 HTMLCollection에 담아 반환
  2. Elemnet.prototype.firstElementChilde : 첫번째 자식 요소 노드 반환
  3. Elemnet.prototype.lastElementChilde : 마지막 자식 요소 노드 반환
