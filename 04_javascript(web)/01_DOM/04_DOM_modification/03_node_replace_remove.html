03. 노드 교체, 삭제

        Node.prototype.replaceChild(newChild, oldChild) 메소드는 
        자신을 호출한 노드의 자식 노드인 oldChilde 노드를 newChild 노드로 교체한다.
        이 때 oldChild는 replaceChild 메소드를 호출한 노드의 자식 노드어야하고 oldChild 노드는 DOM에서 제거된다.

노드 삭제

        Node.prototype.removeChild(child) 메소드는 childe 매개변수에 인수로 전달한 노드를 DOM에서 삭제한다.
        인수로 전달한 노드는 removeChild 메소드를 호출한 노드의 자식이어야 한다.
